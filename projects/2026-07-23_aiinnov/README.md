# aiinnov — 商標データによるAIイノベーション指標

ai-project-loop **Cycle 28** の成果物(2026-07-23)。

## 概要

商標(trademark)レコードから AI 関連を分類し、**AI比率・年次成長率・セクター拡散**
(AI導入セクター数・濃度HHI)を算出するイノベーション指標ツール。AI特許が研究開発を捉える
のに対し、商標は企業が AI を新製品・サービスに実装する動きを全セクター横断で捉える。
標準ライブラリのみ・決定論的。

## 着想元(11_AI Archive)

- [[2026-07-23-measuring-ai-innovation-with-trademark-data-7ec5]] —
  商標データが AI 特許を補完し、企業の AI 実装(新製品・サービス)を**全経済セクター横断・最新**で
  捉えると提案した論文。本ツールはその指標(セクター横断のAI導入・拡散)を集計として実装。

## 使い方

```bash
python make_sample.py                 # デモ用の商標レコード生成
python aiinnov.py sample/marks.csv
```

- 入力: `year,sector,text` の CSV / `--json` で機械可読出力
- AI分類: 単語境界での keyword 照合(「ai」が retail/email 等に誤マッチしない)

## 動作確認結果(2026-07-23)

商標257件(6セクター×2019-2025):

```
  year   AI件数  AI比率   前年比   導入セクター
  2019      1    4.0%     —        1
  2022     11   31.4%   +83%       5
  2025     28   53.8%   +22%       6
-- AI導入セクター数 6  セクター濃度HHI 0.206(低いほど全セクターへ広く拡散)
```

AI商標比率が 4.0%→53.8% へ伸び、導入セクターが 1→6 に拡散(software 先行→全セクターへ)。
成長率は基数増加で減速する S 字を描き、HHI 0.206 で広い普及を示す。商標が捉える
「AI実装の全セクター横断的普及」を定量化できている。

## 制限事項

- AI分類は keyword 照合(意味的分類・商標のニース分類コードは未使用)。誤分類は残る
- サンプルは合成データ。実運用は各国商標庁のオープンデータ(年・区分・名称)を投入する想定
- 指標は件数ベース。商標の質・存続・企業規模による重み付けは扱わない
