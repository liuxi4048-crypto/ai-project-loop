# copyrate — 推論トレースの逐語コピー率(copy-rate)計測器

ai-project-loop **Cycle 6** の成果物(2026-07-23)。

## 概要

LLMの推論トレースが、入力文脈からどれだけ**逐語コピー**しているかを測るCLI。
トレース中のトークンのうち、文脈から連続 `MIN_SPAN` トークン以上の一致で覆われる割合
(copy-rate)を算出する。高いほど「根拠に基づく推論」でなく「貼り付け」寄り。
Python 3 標準ライブラリのみ。

## 着想元(11_AI Archive)

- [[2026-07-23-copy-less-ground-more-overcoming-repetitive-copying-in-long-d725]] —
  長文脈で推論するLLMが入力をそのまま推論過程へコピーする「反復コピー(repetitive copying)」を
  発見し、フロンティア級で普遍・文脈長が伸びるほど悪化すると報告した論文。本ツールはその失敗を
  **測る**側の診断器(論文は緩和手法の提案、本ツールは検出指標)。

## 仕組み

- トークン列に正規化し、トレースの各位置から「文脈に連続部分列として現れる最長スパン」を貪欲に探す
- 長さ `MIN_SPAN`(既定8)以上のスパンを逐語コピーとみなし、覆われたトークン数 / 総トークン数 = copy-rate
- トークン境界は空白パディングで保証(部分トークン一致を防ぐ)

## 使い方

```bash
python copyrate.py <context_file> <trace_file>
```

- `--min-span 8` / `--cap 200`(1スパン探索上限) / `--json`
- 終了コード: copy-rate ≥ 0.30(高コピー)=1・未満=0 → 評価パイプラインのゲートに使える

## 動作確認結果(2026-07-23)

同じ文脈 `context.txt` に対し、貼り付け寄り・言い換え寄りの2トレースで明確に分離:

```
trace_copy.txt      copy-rate 66.0%  ⚠ 高コピー   copied 31/47  最長 19 tokens
trace_grounded.txt  copy-rate  0.0%  ○ 低コピー   copied 0/55   最長 0 tokens
```

コピー版は文脈の2文をほぼ丸写し(19トークン連続一致)、言い換え版は逐語一致ゼロ。
論文の「反復コピー」現象を定量的に切り分けられている。

## 制限事項

- 逐語(表層)一致のみ。言い換えコピー・語順入替は検出しない(意味的類似は対象外)
- 最長一致は貪欲・線形拡張(最悪 O(トレース長 × 文脈長))。超長文脈では低速になり得る
- しきい値0.30は目安。タスクによって適正値は変わる
