# distrl — カテゴリカル分布Bellmanバックアップ(分布強化学習)

ai-project-loop **Cycle 31** の成果物(2026-07-23)。

## 概要

分布強化学習は、期待収益(スカラー価値)ではなく**収益そのものの分布** Z(s) を学習する。
分布的Bellman作用素 `T Z(s) = R(s) + γ Z(s')` を、固定 atom 台への射影(C51流)で反復適用し、
Z(s) が真の収益分布へ収束する様子を **Cramér距離**(CDFのL2)で測る。分布平均が価値関数 V(s) と
一致することも検証する。標準ライブラリのみ・決定論的。

## 使い方

```bash
python distrl.py --atoms 11 --vmax 10 --gamma 1.0
```

- `--json` で機械可読出力

## 動作確認結果(2026-07-23)

MRP: A →(0.5)B→報酬10→終端 / A →(0.5)C→報酬0→終端(γ=1):

```
状態AへのCramér距離(反復ごと): 1.581 → 0.000 → 0.000 → …(不動点へ収束)
分布平均 = 価値関数 V: {A:5.0, B:10.0, C:0.0, T:0.0}  (解析解と一致)
Z(A) の収益分布(不動点):
  return  0.0: 0.50 ███████████████
  return 10.0: 0.50 ███████████████
```

Z(A) は真の二峰分布(0と10が各0.5)へ収束し、その平均は V(A)=5 に一致。分布的Bellman
バックアップと Cramér 距離による収束、分布平均=価値関数の整合を正しく再現できている。
