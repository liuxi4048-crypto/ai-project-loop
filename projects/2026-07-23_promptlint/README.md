# promptlint — プロンプトの指示遵守/ハルシネーション リスクリンター

ai-project-loop **Cycle 12** の成果物(2026-07-23)。

## 概要

プロンプト文を解析し、**書式・指示数・文脈長の3因子**からリスクを採点して具体的な改善警告を
出す静的リンター。「その3因子が指示遵守率とハルシネーションを左右する」という実証研究の
知見を、モデル不要の事前チェックに落とし込んだもの。Python 3 標準ライブラリのみ・決定論的。

| 因子 | 見るもの | しきい値(ヒューリスティック) |
|------|----------|------------------------------|
| instruction_count | 箇条書き/番号/命令文・助動詞(must/should)の数 | 10超で上昇・30で飽和 |
| context_length | 粗いトークン推定 | 4000超で上昇・16000で飽和(中盤忘却帯) |
| format | 混在する構造フォーマット数(bullet/numbered/table/xml/json) | 2種以上で上昇 |

## 着想元(11_AI Archive)

- [[2026-07-23-prompt-design-at-scale-how-format-instruction-count-and-cont-975f]] —
  汚染フリー合成コーパスで5モデルを統制実験し、**フォーマット・指示数・文脈長**の3因子が
  指示遵守率とハルシネーションに与える影響を定量化した論文。本ツールはその3因子を実務向けの
  事前リントに落とし込んだもの。

## 使い方

```bash
python promptlint.py sample/risky.txt
```

- `--json` で機械可読出力 / 終了コード: 高リスク(score≥60)=1・未満=0

## 動作確認結果(2026-07-23)

```
clean.txt   risk 0.0 /100  [低リスク]   指示4 / 75tok / md+bullet     → 警告なし
risky.txt   risk 76.2/100  [高リスク]   指示31 / 7842tok / 4形式混在
  - 指示数 31(>25)→ 遵守率低下の恐れ
  - 推定 7842 トークン → 重要指示は中盤に埋めない
  - 構造フォーマット混在(numbered/table/xml/json)→ 1つに統一
```

素直なプロンプトと、指示過多・長文脈・フォーマット混在のプロンプトを明確に区別できている。

## 制限事項

- しきい値は研究の定性的知見に基づくヒューリスティック(モデル・タスク非依存の絶対基準ではない)
- 指示数・トークン数は正規表現ベースの近似(正確なトークナイザ計数ではない)
- 英語想定の命令形辞書。日本語プロンプトの命令文抽出は限定的
