# conformal-coverage — 共形予測のクラス別カバレッジ崩壊デモ

ai-project-loop **Cycle 4** の成果物(2026-07-23)。

## 概要

分割共形予測(split conformal prediction)が、**周辺カバレッジ 1-α は保つのにクラス別
カバレッジは大きくばらつく**こと、そして**分布シフト下では周辺カバレッジ自体も割れる**ことを、
合成データで実演するCLI。Python 3 標準ライブラリのみ・乱数シード固定で再現可能。

## 着想元(11_AI Archive)

- [[2026-07-22-the-label-complexity-of-class-conditional-coverage-under-dis-6834]] —
  「分割共形予測は分布シフト下で全体カバレッジを保つがクラス別カバレッジは低下する」という
  ラベル複雑性の論文。本ツールはその主結果を最小構成で再現する教材。

## 仕組み

- 非適合スコア = `1 - p(真ラベル)`。クラスごとにスコア品質(平均確率)を変えてある
- 閾値 `q̂` = 較正集合の `⌈(n+1)(1-α)⌉` 番目に小さい非適合スコア(全クラス共通の1つ)
- 「非適合スコア ≤ q̂ なら真ラベルを予測集合に含む」→ カバレッジを全体・クラス別に集計

## 使い方

```bash
python conformal.py
```

- `--alpha 0.1`(目標カバレッジ=90%)/ `--n-cal` / `--n-test` / `--seed`

## 動作確認結果(2026-07-23, seed=0, α=0.1)

```
=== シフトなし(較正=評価=均等) ===
 class  quality   カバレッジ   目標との差
     0     0.85     100.0%     +10.0%  ← 崩れ
     1     0.70      99.3%      +9.3%  ← 崩れ
     2     0.55      92.3%      +2.3%
     3     0.40      70.1%     -19.9%  ← 崩れ
周辺カバレッジ = 90.6%  /  クラス別カバレッジの幅 = 29.9%

=== 分布シフトあり(評価は低品質クラスに偏る) ===
周辺カバレッジ = 85.3%  /  クラス別カバレッジの幅 = 32.2%
```

- シフトなし: **周辺は90.6%で目標どおり**なのに、クラス別は70%〜100%と約30ポイント開く
  (単一閾値が全クラスを混ぜるため、質の低いクラス3が過小カバレッジ)
- シフトあり: 交換可能性が崩れ、**周辺カバレッジ自体も85.3%へ低下**

論文の主張(全体は保つがクラス別が崩れる/シフトで悪化)を再現できている。

## 制限事項

- スコアは合成モデル(クラス別ガウス)。実分類器の logits ではない
- 真ラベルのカバレッジのみ評価(予測集合サイズや APS 等の高度なスコアは未実装)
- クラス別カバレッジを回復する手法(Mondrian/クラス別共形など)は含まない — 問題の提示まで
