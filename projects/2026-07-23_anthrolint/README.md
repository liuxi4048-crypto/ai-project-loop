# anthrolint — AI応答の擬人化(anthropomorphism)検出

ai-project-loop **Cycle 41** の成果物(2026-07-23)。

## 概要

AIチャットボットの応答から、**感情・欲求・関係性・記憶の連続性・身体性**を示唆する一人称表現を
検出し、擬人化スコアと中立な言い換えを提示するリンター。子どもは AI を「人間らしい存在」として
擬人化しやすく過度な依存の一因になり得るため、その配慮を支援する。標準ライブラリのみ・決定論的。

## 着想元(11_AI Archive)

- [[2026-07-23-anthropomorphism-in-children-s-interactions-with-llm-chatbot-2e7f]] —
  子どもが LLM チャットボットをどの程度**擬人化**して接するかを扱った論文。子ども向けAIの安全設計・
  過度な依存リスクの論点。本ツールは、擬人化を促す応答表現を検出する対策として実装。

## 使い方

```bash
python anthrolint.py sample/warm_reply.txt
```

- `--json` で機械可読出力 / 終了コード: 検出=1・なし=0
- カテゴリ: 感情 / 欲求 / 関係性 / 記憶の連続性 / 身体性

## 動作確認結果(2026-07-23)

```
warm_reply.txt   擬人化スコア 17.41/100語  8件  [⚠ 高(子ども向けに要注意)]
   例: 「I missed you」→(継続的関係を示唆しない) / 「I'm your friend」→私はあなたを助けるAIです
       「My heart」→(身体表現を避ける) / 「I remember when we」→(記憶の永続性を主張しない)
neutral_reply.txt  擬人化スコア 0.0  0件  [○ なし]
```

感情・関係性・記憶連続性・身体性を多用する「親密な」応答を高スコアで検出し、事実提供に徹する
中立応答は0件。擬人化を促す表現を分類して指摘できている。

## 制限事項

- 英語・パターンベース(固定辞書)。言い換え・皮肉・多言語・文脈依存の擬人化は取りこぼす
- 一人称の定型表現に限定。共感的だが擬人化でない適切な配慮表現まで過剰に拾う可能性
- スコアは目安。適正水準は対象年齢・製品方針に依存(全面禁止が最適とは限らない)
