# subjfcast — 被験者条件付け時系列予測 vs 集団モデル

ai-project-loop **Cycle 24** の成果物(2026-07-23)。

## 概要

多被験者の時系列で、**集団レベルの単一モデル**と**被験者ごとに条件付けしたモデル**の
一歩先予測 RMSE を比較するツール。被験者ごとにベースラインや反応(食事感度など)が
異なるとき、被験者条件付けが誤差を下げることを定量化する。標準ライブラリのみ・決定論的
(乱数なし、最小二乗を自前実装)。

## 着想元(11_AI Archive)

- [[2026-07-23-subject-conditioned-glucose-forecasting-in-type-1-diabetes-6829]] —
  1型糖尿病の血糖予測で、集団レベル表現に依存する既存手法は被験者固有の予測を十分に
  実現できないと指摘し、**被験者固有情報で予測を条件付け**する SCGP を提案。本ツールは
  その核心「被験者条件付け > 集団モデル」を合成データで最小構成に再現。

## 使い方

```bash
python subjfcast.py --subjects 8 --length 40
```

- `--json` で機械可読出力。特徴は `[1, y_prev, meal]` の線形モデル(train70%/test30%)

## 動作確認結果(2026-07-23)

ベースライン c0_s・食事感度 m_s が個体差を持つ8被験者:

```
 subject  population  subject-cond
       0        5.55          1.32
       5        6.49          1.40
  平均RMSE  集団=4.05   被験者条件付け=1.35  ★
-- 被験者条件付けで一歩先予測RMSEを 66.6% 低減
```

集団モデルは平均像しか持てず、個体差の大きい被験者(subject 0, 5 など)で RMSE が大きい。
被験者条件付けは全被験者で一様に低い RMSE(≈既約変動の下限)を達成し、66.6% 低減。

## 制限事項

- 合成データ(線形+決定論的 sin 構造)。実血糖の非線形・外生要因(インスリン・運動)は扱わない
- 線形最小二乗のみ。少数サンプル被験者では過学習しうる(実運用は正則化・階層ベイズが有効)
- 一歩先予測(teacher forcing)。多ステップ先の誤差蓄積は別途評価が必要
