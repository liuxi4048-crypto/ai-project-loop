# lqr — 離散時間LQR最適制御

ai-project-loop **Cycle 37** の成果物(2026-07-23)。

## 概要

線形系 `x_{t+1}=A x_t + B u_t` を二次コスト `J=Σ(xᵀQx + uᵀRu)` 最小で制御する
**LQR(線形二次レギュレータ)**。離散代数リッカチ方程式を反復で解いて最適フィードバック
ゲイン K を求め、`u=-Kx` で閉ループ制御する。標準ライブラリのみ・決定論的
(小行列演算とリッカチ反復を自前実装)。

## 着想元(11_AI Archive)

- [[2026-07-23-real-time-optimal-control-with-shallow-recurrent-decoder-net-8877]] —
  低次元化モデルで高次元ダイナミクスの**閉ループ制御器を実時間合成**するリアルタイム最適制御の
  研究。本ツールはその古典的中核であるLQRを実装し、リッカチ解による最適フィードバックを再現。

## 使い方

```bash
python lqr.py --steps 60 --r 0.1
```

- `--json` で機械可読出力。系は二重積分(位置・速度、制御=加速度)

## 動作確認結果(2026-07-23)

初期状態 位置1・速度0 から:

```
  最適ゲイン K = [2.585, 3.575]   (u = -K·x)
  状態ノルム |x|: 1.03 → 0.85 → 0.46 → 0.22 → 0.11 → … → 0.004(0へ収束)

            制御方式    総コスト   最終|x|
         LQR(最適)     13.827    0.004  ★
         素朴な比例制御   390.050   3.567   ← ゲイン誤設定で不安定化
             無制御      60.000   1.000   ← 減衰せずコスト累積
```

LQRは状態を0へ収束させつつ**総コスト最小**。位置のみに固定ゲインを与える素朴な比例制御は
この系を不安定化させ、無制御は状態が減衰せずコストを積む。リッカチ解の最適性を確認できている。

## 制限事項

- 線形時不変系・二次コストの LQR(非線形・制約付きは iLQR/MPC が必要)
- 状態は完全観測を仮定(部分観測は LQG=LQR+カルマンフィルタが必要)
- リッカチは反復解法(値反復)。大規模系は専用のリカチ解法が望ましい
