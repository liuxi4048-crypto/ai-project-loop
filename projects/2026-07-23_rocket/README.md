# rocket — ランダム畳み込み特徴による時系列分類(ROCKET系)

ai-project-loop **Cycle 38** の成果物(2026-07-23)。

## 概要

多数の**ランダムな畳み込みカーネル**を時系列に当て、各出力から **最大値(max)** と
**正値割合(PPV: proportion of positive values)** の2特徴を取って固定次元(2K)の表形式特徴へ
写像し、その上で線形分類器を学習する ROCKET 系の時系列分類。単純ながら強力。標準ライブラリ
のみ・乱数シード固定で決定論的。

## 着想元(11_AI Archive)

- [[2026-07-23-in-context-time-series-classification-with-random-convolutio-bb4a]] —
  時系列を**ランダム畳み込み変換で固定次元の表形式特徴へ写像**して分類する手法(MASHT)。本ツールは
  その中核である ROCKET 変換(ランダム畳み込み→max/PPV特徴→線形分類)を実装。

## 使い方

```bash
python rocket.py --kernels 100 --length 80 --seed 0
```

- `--json` で機械可読出力

## 動作確認結果(2026-07-23)

周波数の異なる2クラス(0.10 / 0.16、ランダム位相・振幅、ノイズσ=0.5)の合成時系列:

```
ROCKET  カーネル 100本 → 特徴 200次元  系列長 80
  学習 60系列 / テスト 40系列
  訓練精度 100.0%   テスト精度 100.0%(seed 0, 3 とも)
```

ランダム位相・振幅とノイズにより単純な閾値では分離できないタスクを、ROCKET の
シフト不変な畳み込み特徴(max + PPV)+ 線形分類でテスト精度 100% で分類。複数シードで再現。

## 制限事項

- 合成2クラス(周波数差)のデモ。実データ(不等長・多クラス・微妙な局所形状)では精度は下がる
- ロジスティック回帰をGDで学習(ROCKET原論文はリッジ回帰)。特徴標準化は訓練統計で実施
- カーネル数K・長さ・dilationは簡略な既定値。大規模はMINIROCKET等の高速版が望ましい
