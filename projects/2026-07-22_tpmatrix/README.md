# tpmatrix — 全正行列の判定と特性多項式係数

ai-project-loop **Cycle 32** の成果物(2026-07-23)。

## 概要

行列が **全正(totally positive, TP: すべての小行列式が正)** / **全非負(TNN: すべて非負)** かを
判定し(違反する小行列式を提示)、**特性多項式の係数**を算出するCLI。特性多項式
`det(λI−M)=Σ(−1)^k e_k λ^{n−k}` の係数 `e_k` は「位数 k のすべての主小行列式の和」で、
e_1=トレース、e_2=2×2主小行列式和…と続く。標準ライブラリのみ・**厳密(有理数)**・決定論的。

## 着想元(11_AI Archive)

- [[2026-07-22-totally-positive-matrices-and-the-highest-order-coefficients-fccb]] —
  全正行列が**特性多項式の最高次係数**を通じてどの程度区別できるかを調べた論文。本ツールは
  その基礎量(TP/TNN判定と特性多項式係数=主小行列式和)を厳密計算で実装。

## 使い方

```bash
python tpmatrix.py                # 組込み例(全正Vandermonde / 非TP / 単位行列)
python tpmatrix.py matrix.json    # [[...],[...]] を判定
```

## 動作確認結果(2026-07-23)

```
Vandermonde nodes(1,2,3): 全正(TP)  特性多項式 λ³-12λ²+15λ-2  (e1=12,e2=15,e3=2)
[[1,2],[3,1]]:            TP/TNNでない  小行列式 rows[0,1]cols[0,1] = -5
単位行列 I_3:             TNN だが TP でない(rows[0]cols[1]=0)  特性多項式 (λ-1)³
```

いずれも手計算と一致(Vandermondeのdet=2=(2-1)(3-1)(3-2)、trace=12、単位行列は固有値1の三重)。
全正性の判定と、特性多項式の高次係数を与える主小行列式和 e_k を厳密に算出できている。

## 制限事項

- 全小行列式を列挙(位数×行選択×列選択)するため n が大きいと組合せ爆発(小〜中サイズ向け)
- 行列式は余因子展開(小サイズ厳密向け)。大規模は分数フリー消去(Bareiss)等が必要
- 判定は厳密有理数。浮動小数入力は Fraction 変換時の丸めに注意
